# 🌊 Beach Monitoring Dashboard

> Sistema de monitoreo en tiempo real para la gestión operativa de playas
> Diseñado desde la lógica de un socorrista, no desde un escritorio

---

## 🧠 Contexto

En una playa, las decisiones no pueden esperar.
El estado del mar cambia, el viento rota, aparece una corriente… y en segundos tenés un incidente.

Este proyecto nace con una idea clara:

👉 Centralizar información crítica en una interfaz simple, rápida y accionable.

No es solo un dashboard. Es una herramienta pensada para uso real.

---

## 🚀 Qué hace

* 🌊 **Estado del mar**
  Sistema visual tipo semáforo (verde / amarillo / rojo)

* 🌬️ **Condiciones climáticas**
  Viento, temperatura y variables clave

* 🚨 **Sistema de alertas**
  Corrientes peligrosas, medusas, bandera roja

* 🧍‍♂️ **Gestión de incidentes**
  Registro rápido de eventos en playa

* 🗺️ **Zonificación**
  División por sectores de cobertura

## 🧠 Decisiones de diseño

* UI simple → pensada para uso bajo estrés
* Información priorizada → lo crítico primero
* Interacción rápida → mínimo de clicks
* Visual → estados claros en segundos
